Pre Order

  • Website orders can be pre-ordered.
  • Use 'Enable Pre-Order' field in Settings > Sales > Quotations & Orders for turning On/Off this feature.
  • If the feature is enabled orders placed will be shown for accepting just before it needs to be done.
  • Website should block completing the pre-order if time remaining to finish order preparation is less than time required to prepare the order.

Preparation Time Calculation

Calculation:

Time required to prepare the order is the sum of product of preparation time for each product and its quantity.
  • Eg:
    • Preparation time for a product is 10 minutes
    • Order line quantity is 2
    • Preparation time for second product is 20 minutes
    • Order line quantity is 3
    • So the total preparation time for the order will be:
      • (10 x 2) + (20 x 3) = 80 minutes
  • For preparation time for a product use 'Preparation time in minutes' field in Product Variants if it is greater than 0.
  • Else use 'Preparation time in minutes' field in Products if it is greater than 0.
  • If both field are zero then last option is to take the default preparation time for products. Use 'Minimum preparation time in minutes' field in Settings > Inventory > Products.
